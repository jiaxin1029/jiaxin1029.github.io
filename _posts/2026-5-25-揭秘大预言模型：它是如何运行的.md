你有没有想过，当你给 ChatGPT、Claude 或 Llama 发送一条消息时，幕后究竟发生了什么？它是像人类一样在思考，还是只是一个高级的“复读机”？

今天，我们不谈枯燥的数学公式，用最直观的方式拆解 大语言模型（LLM）的核心运行机制。

核心本质：地表最强的“接龙游戏”
如果用一句话概括 LLM 的本质，那就是：根据已经上文，预测下一个最可能出现的单词（Token）。

是的，它在做的其实就是文字接龙。


![LLM是如何工作的](blob:https://gemini.google.com/d95ec5e2-cb45-4512-9ca8-978a6837b566)

当你输入：“床前明月光，”

模型根据训练学到的知识，计算出下一个字是“疑”的概率最高（比如 99%）。

于是它输出“疑”，接着把“床前明月光，疑”作为新的输入，继续预测下一个字“是”……以此类推。

LLM 运行的四大核心步骤
从你的输入到模型的输出，通常需要经历以下四个标准工作流：

--1. **分词** （Tokenization）—— 把人类语言切成“方块”
AI 是看不懂人类高级语言的，它只能处理数字。所以第一步是把你的句子切碎成一个一个的“Token”（词元）。

例子："我喜欢机器学习" 可能会被切分为 ["我", "喜欢", "机器", "学习"]。

每个 Token 在模型的字典里都有一个独一无二的数字 ID。

--2. **向量化** （Embedding）—— 赋予文字“空间坐标”
单纯的数字 ID 无法体现词语之间的关系（比如“猫”和“狗”很像，但和“汽车”相差很远）。

向量化技术把每个 Token 变成一串包含几千个数字的高维向量。

在这个高维空间里，语义相近的词会被放在相邻的位置。

--3. **注意力机制** （Attention）—— 读懂上下文
这是 LLM 拥有“高情商”和“理解力”的核心所在（即 Transformer 架构）。

传统的机器翻译是一个词一个词硬翻，而注意力机制（Self-Attention）让模型在处理每一个词时，都会去抬头看一眼句子里的其他词。

经典例子："苹果今天发布了新手机" 和 "我很喜欢吃苹果"。在这两个句子里，“苹果”这个词通过注意力机制，会分别和“手机”或“吃”产生强关联，从而让模型准确判断出它的真实含义是“科技公司”还是“水果”。

--4. **采样与生成** （Sampling & Generation）—— 概率的艺术
经过层层计算，模型最终会给词表里的几十万个词各分配一个概率。

如果模型永远只选概率最高的那个词（称为 Greedy Search），它的回答会变得非常机械和重复。

为了让回答更自然、更有创意，模型会引入 Temperature（温度） 等参数进行随机采样：

低温度（如 0.2）：模型非常保守，只选概率最高的，适合写代码或回答客观事实。

高温度（如 0.8）：模型更愿意冒险，会选一些概率稍低的词，适合写小说或脑暴。

为什么 LLM 会出现“幻觉”？
理解了运行机制，你就很容易明白为什么 AI 有时候会“一本正经地胡说八道”（幻觉）了。

因为 LLM 内部并没有一个客观世界的“真理数据库”。它所做的一切，都是基于概率的统计拟合。当它不知道正确答案时，为了完成“接龙”的任务，它会根据概率计算出一串听起来最像人话、逻辑最顺畅，但实际上完全虚构的字符串。

总结：它是一面镜子
LLM 没有灵魂，也没有真正的意识。它更像是一面镜子，通过数万亿字的人类文明语料，折射出了人类语言的规律与逻辑。我们输入提示词（Prompt）的过程，就像是在调整镜子的角度，引导它照出我们想要的答案。
