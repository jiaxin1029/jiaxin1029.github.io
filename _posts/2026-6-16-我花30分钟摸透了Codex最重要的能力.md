# 我花30分钟摸透了Codex最重要的能力

> 如果你还把 Codex 当成一个会写代码的 kimi或者Chatgpt，那么你可能只用了它 5% 的能力。

最近看完《Learn 95% of Codex in 30 Minutes》后，我最大的感受是：

**Codex 根本不是聊天机器人，而是一个真正的 AI Agent 工作系统。**

它不仅能回答问题，还能：

* 读取和修改文件
* 操作浏览器
* 控制电脑
* 连接第三方工具
* 自动执行工作流
* 持续记忆你的习惯

简单来说：

> ChatGPT 帮你思考，Codex 帮你干活。

本文将用最短时间带你理解 Codex 最重要的 7 大能力。

---

# 什么是 Codex？

很多人第一次接触 Codex 时，以为它只是一个更擅长编程的 AI。

实际上：

```text
ChatGPT = AI 助手

Codex = AI 员工
```

它最大的区别在于：

不仅会生成内容，

还能直接执行任务。

这也是为什么越来越多人把它称为：

> 「真正意义上的 AI Agent」

---

# 1. Full File Access（完整文件访问）

这是最容易被低估的能力。

传统 AI：

```text
上传文件
↓
分析文件
↓
结束
```

Codex：

```text
读取文件
↓
修改文件
↓
创建文件
↓
整理文件
↓
输出结果
```

例如：

```text
分析我下载目录里的所有发票
并生成Excel报表
```

Codex 可以自动：

* 查找文件
* 提取数据
* 汇总统计
* 创建表格
* 输出图表

整个过程无需人工干预。

这意味着：

很多重复性办公工作都可以直接交给 AI。

---

# 2. Persistent Memory（持久记忆）

真正强大的 AI 一定会记住你。

Codex 提供两种记忆方式。

## Manual Memory

手动记忆

例如：

```text
以后所有营销文案
都采用极简科技风
```

Codex 会长期遵循这一偏好。

---

## Auto Memory

自动记忆

随着使用时间增加：

Codex 会逐渐学习：

* 你的写作风格
* 项目习惯
* 常用工具
* 工作流程

最终形成专属于你的工作模式。

这就像培养一个越来越懂你的员工。

---

# 3. Plugins（插件生态）

如果说 ChatGPT 是一个大脑，

那么插件就是它的手脚。

Codex 可以连接：

* Gmail
* Slack
* Notion
* Google Drive
* GitHub

等等。

例如：

```text
分析最近两周邮件
寻找潜在合作机会
```

Codex 可以：

* 登录邮箱
* 阅读邮件
* 提取关键信息
* 输出分析报告

全程自动完成。

---

# 4. Skills（最被低估的能力）

如果你只能学会一个功能，

我建议学 Skills。

因为它决定了：

你是在使用 AI，

还是在训练 AI。

---

## 什么是 Skill？

Skill 本质上是：

```text
经验 + SOP + 自动化
```

例如：

你花了两小时做出一个：

```text
小红书爆款选题分析流程
```

效果非常好。

此时可以保存为：

```text
Skill
```

以后只需要输入：

```text
分析本周AI热点
```

Codex 就会自动执行整套流程。

---

## 普通用户

```text
每次重新写提示词
```

## 高级用户

```text
把流程沉淀成Skill
```

然后无限复用。

这才是真正的效率革命。

---

# 5. GPT Image（图片生成）

过去生成图片需要：

```text
ChatGPT
↓
Midjourney
↓
Photoshop
```

现在：

```text
Codex
```

一步完成。

例如：

```text
为我的品牌生成5张广告图
```

Codex 可以：

* 创建图片
* 自动保存
* 分类管理

大幅减少切换工具的时间。

---

# 6. Browser Use + Computer Use

这是 Agent 最核心的能力。

---

## Browser Use

控制浏览器。

例如：

```text
测试我的网站
```

Codex 会：

* 打开网站
* 点击按钮
* 检查页面
* 寻找Bug

就像真实用户一样操作。

---

## Computer Use

控制电脑。

例如：

```text
帮我制作产品PPT
```

Codex 可以：

* 打开软件
* 导入素材
* 自动排版
* 输出成品

未来很多基础办公工作都将被这种能力彻底改变。

---

# 7. Automations（自动化）

这是打工人的终极武器。

例如：

```text
每周五上午9点
自动生成市场分析报告
```

Codex 会定时执行。

无需提醒。

无需手动操作。

适用于：

* SEO日报
* 竞品监控
* 数据统计
* 邮件汇总
* 内容运营

---

# Bonus：Chronicle

视频最后提到的隐藏功能。

Chronicle 的作用是：

持续理解你的工作上下文。

例如：

你正在制作一份商业计划书。

Codex 会主动发现：

```text
缺少市场分析
缺少竞品研究
缺少财务预测
```

然后给出建议。

这已经不是工具，

更像是一位随时在线的助理。

---
